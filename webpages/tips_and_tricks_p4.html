Tips and Tricks with Jupyter Notebooks¶

Part 4: Creating Interactive dashboards¶

Building interactive dashboards in Jupyter Notebooks can potentially be a long and complicated process.

Top Tip: Understand your requirements

It may be ok for small projects just for you. However, it may be quicker to use a data visualization tool like PowerBI or Tableau.

  • It will use less coding
  • They can provide a more shareable solution
  • You don't have to worry about installing requirements

If you are looking to collaborate with notebooks in a shared environment, consider a cloud solution like Databricks.

  • Remember, interactive dashboards in Jupyter won't work if you export them as static html files

Typical libraries for Jupyter Dashboards¶

  • Pandas, Numpy
  • SciKit-Learn - models
  • Matplotlib, Seaborn, Plotly - data visualization
  • ipywidgets e.g. for dropdowns, sliders etc.

Dashboard example: Google Analytics Linear Regression Project¶

My linear regression project presents some web traffic data in an interactive dashboard.

This is split into two sections:

Section A

  • Shows Graphical representation for Linear Model based on training data using Seaborn.
  • Predictor box provided so the number of Users can be entered to predict Sessions and Page Views.

Section B

  • Provides an overview on web metrics, built using plotly which offers interactive elements for graphs beyond dropdown ipywidgets

This can been constructed with classes for controllers and views

Steps for Accessing the Dashboard on Kaggle¶

  1. Click this link: https://www.kaggle.com/kremzeeq/ga-linear-regression-analysis-dashboarding/edit

  2. Click the blue Copy and Edit button

  3. You have a Kaggle account, you can login and upload the GA_web_traffic_data_12_08_20.csv file.

  4. Next, you should be able to click 'Run All' for the notebook with the dashboard and interact with notebook widgets.
